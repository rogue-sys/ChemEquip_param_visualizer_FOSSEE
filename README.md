# Chemical Equipment Parameter Visualizer (Hybrid Web + Desktop)

A hybrid analytics application designed to visualize and analyze chemical equipment parameters from CSV files. The system provides a unified backend with both a web interface and a desktop application, enabling users to upload datasets, view summaries, explore visualizations, and manage upload history.

## Tech Stack

**Backend**
- Python
- Django
- Django REST Framework
- Pandas
- SQLite

**Web Frontend**
- React.js
- Chart.js
- Axios

**Desktop Application**
- PyQt5
- Matplotlib
- Requests

**Reporting**
- PDF Generation (Python)

## Features

- CSV upload and parsing for chemical equipment data
- Automated data analysis and summary statistics
- Interactive visualizations on the web (Chart.js)
- Desktop-based visual analytics using Matplotlib
- Token-based user authentication
- Per-user upload history (last 5 datasets)
- Summary-based PDF report generation
- Shared backend for both web and desktop clients

This project demonstrates a real-world hybrid application architecture with consistent data handling and visualization across multiple client platforms.
